>**Objetivo da Disciplina:** 
> Formar uma base sólida em programação através de três pilares fundamentais:

1. CONCEITOS BÁSICOS

- Introdução à lógica de programação
- Entendimento de algoritmos
- Compreensão de estruturas básicas de dados
- Familiarização com pseudocódigo

2. DESENVOLVIMENTO DO RACIOCÍNIO LÓGICO

- Capacidade de decompor problemas
- Pensamento sequencial e estruturado
- Criação de soluções sistemáticas
- Análise lógica de situações

3. CAPACIDADE DE IMPLEMENTAÇÃO

- Transformar problemas em [[Algoritmos]]
- Escrever pseudocódigo funcional
- Desenvolver soluções práticas
- Criar programas básicos

HABILIDADES ESPERADAS AO FINAL DA DISCIPLINA:

- Analisar problemas de forma estruturada
- Criar soluções algorítmicas eficientes
- Implementar códigos básicos
- Compreender a base para programação avançada
- Desenvolver pensamento computacional

Esta disciplina serve como fundamento essencial para:

- Aprendizado de linguagens de programação
- Desenvolvimento de sistemas
- Compreensão de estruturas de dados
- Evolução profissional na área de tecnologia

MÉTODO DE APRENDIZADO:

- Teoria combinada com prática
- Exercícios progressivos
- Desenvolvimento de projetos
- Uso de ferramentas como Visualg
- Prática constante de resolução de problemas
